# Gatsby 시도하기

## 프로젝트 설정

1. 설치
   `npm i -g gatsby-cli`

2. 세팅
   `gatsby new`
   `cd _프로젝트 폴더_`

세팅시 TypeScript를 들고 시작 할 수 있어서 편리하다!
버전 확인시 `gatsby --version`

3. 로컬 개발 서버 실행

> `npm run develop` ->
> `http://localhost:8000/`사이트이며, 개발모드에서 구동중
> cra와 동일

> `npm run build` -> `npm run serve` ->
> `http://localhost:9000/`으로 빌드가 되며 페이지내 콘텐츠가 이미 생성되어있다.
